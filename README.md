# IoT-Enabled Smart Agriculture Water Controller

## Overview

The IoT-Enabled Smart Agriculture Water Controller is a smart irrigation system built using ESP8266 (NodeMCU). It monitors soil moisture, temperature, and humidity in real time and provides a web-based dashboard for remote monitoring and water pump control over Wi-Fi. The system helps improve water management by enabling users to monitor environmental conditions and operate irrigation from any device connected to the same network.

## Features

- Real-time temperature and humidity monitoring
- Live soil moisture monitoring
- Remote water pump ON/OFF control
- Wi-Fi-enabled web dashboard using ESP8266
- Live sensor updates with an interactive dashboard
- Mobile and desktop compatible interface
- Simple, lightweight, and easy to deploy

## How It Works

1. Open the project in Arduino IDE
2. Update the Wi-Fi hotspot name and password in the source code
3. Upload the code to the ESP8266 board
4. Open the Serial Monitor and note the IP address displayed after a successful Wi-Fi connection
5. Enter the IP address in any web browser connected to the same network
6. The dashboard displays live temperature, humidity, soil moisture, and pump status
7. Control the water pump directly from the dashboard
